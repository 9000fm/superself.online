# SUPERSELF — Brand Reference

## Identity

**SUPERSELF** is an electronic music label and creative collective (est. 2023).
We share mixes, releases, and curated recommendations. We connect artists with
curious ears and make space for sounds that deserve to be heard. Sound first.

## Visual Language

### Logo
The logo is a **perspective grid room** with a **floating white sphere** at the center.
It represents depth, structure, and a world within — looking into infinite space.

- Primary logo: `public/superself-logo-wh.webp` (white on transparent, 800x800)
- Logo variants: `public/logos/` (colored versions: azul, blanco, negro, rojo, amarillo, morado)

### Colors
| Token | Value | Usage |
|-------|-------|-------|
| Primary | `#0000FF` | Accent, selection, links, interactive elements |
| Background (dark) | `#000000` | Default mode — dark, minimal |
| Background (light) | `#0000FF` | Alt mode — electric blue |
| Foreground | `#FFFFFF` | Text, lines, borders |
| Win95 Grey | `#C0C0C0` / `#3A3A3A` | UI chrome (light/dark mode) |

### Typography
- **Terminal/DOS**: VT323 (Google Fonts) — primary display font
- **Win95 UI**: "MS Sans Serif", Arial — dialog text, buttons
- **Monospace fallback**: Consolas, "Courier New"

### Aesthetic Principles
1. **Minimal** — Less is more. Every element earns its place.
2. **Architectural** — Grid-based, structured, geometric perspective.
3. **Serious** — Not playful, not gimmicky. Confident and clean.
4. **Digital/Retro-futuristic** — DOS/Win95 DNA, but elevated. Not nostalgic — intentional.
5. **Dark-first** — Black backgrounds, white elements, blue accents.
6. **Interactive but subtle** — Interactions should feel alive, not like toys.

### What SUPERSELF is NOT
- Not vaporwave (no pastels, no irony)
- Not glitch art (no broken aesthetics)
- Not maximalist (no visual noise)
- Not cute (no emojis, no rounded corners, no soft colors)
